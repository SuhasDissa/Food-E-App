## ✨ 特徴
- 560以上の食品添加物のコレクション
- 添加物名またはEナンバーで検索
- 各添加物の詳細
- 食品のバーコードをスキャンして栄養成分情報を取得します。
- アプリ内で食品を検索します。
- 複数の言語をサポート。
- 添加物情報の自動翻訳
- 大画面タブレットをサポート
